# BlackJack-Java-Game
BlackJack Game In Java 


A simple Java Swing Blackjack game! Play against the dealer with Hit, Stand, and Restart buttons. Designed to be minimal, fun, and easy to understand. Perfect for beginners or anyone who wants a small Java GUI project.

Features of my application: 
1) Player vs Dealer gameplay

2) HIT/ STAND / RESTART buttons

3) Auto-calculated totals for player and dealer

4) Detects BUST, WIN, DRAW, and LOSE

5) Minimal GUI with neon-style buttons and totals display

6) Easy to expand (add images, card decks, betting system, etc.)

**   How to play  **
---------------------
HIT – draw a card

STAND – let the dealer play and reveal totals

RESTART – reset the game

**  Rules **
--------------------
Try to get as close to 21 as possible without going over

Dealer hits until they reach 17 or more

Highest total ≤ 21 wins

Tech Used :  Java (JDK 21, NetBeans IDE, Swing)
